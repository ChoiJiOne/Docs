<br><br>


## 문서를 작성하는 절차

개인 문서를 작성하기 위한 절차는 다음과 같습니다.  
<br><br>


### 1단계

markdown 포멧으로 문서를 작성합니다. 이때, 작성자는 적지 않아도 되지만, 문서 번호는 반드시 남겨야 합니다. 그리고 마크 다운 파일의 이름은 문서 번호로 지정해야 합니다.  

<br><br>


### 2단계

문서 작성 시 사용하는 툴은 `Visual Studio Code` 와 `WSL2` 이므로, bash 셸에서 `$ mkdocs build` 명령어를 실행합니다.  

<br><br>


### 3단계


bash 셸에서 `$ mkdocs gh-deploy` 명령어를 실행합니다.  

<br><br>


### 4단계

bash 셸에서 `$ mkdocs serve` 명령어를 실행하여 작성한 문서를 확인합니다. 이 단계는 반드시 필요한 단계는 아닙니다.  

<br><br>


### 5단계

작성한 문서를 commit 하고 원격 저장소에 push 합니다.  

<br><br>
