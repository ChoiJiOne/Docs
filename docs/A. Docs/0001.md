## 문서를 작성하는 절차

개인 문서를 작성하기 위한 절차는 다음과 같습니다.  

- 1단계) markdown 포멧으로 문서를 작성합니다.
- 2단계) bash 셸에서 `$ mkdocs build` 명령어를 실행합니다.
- 3단계) bash 셸에서 `$ mkdocs gh-deploy` 명령어를 실행합니다.
- 4단계) bash 셸에서 `$ mkdocs serve` 명령어를 실행하여 작성한 문서를 확인합니다.
- 5단계) 작성한 문서를 commit 하고 원격 저장소에 push 합니다.
