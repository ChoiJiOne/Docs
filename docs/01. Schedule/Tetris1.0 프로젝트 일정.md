# Tetris1.0 프로젝트 일정
- 작성자 : 최지원
<br><br>


## 기존에 구현한 2D Tetris 게임도 있는데, 다시 하는 이유는?

사실, 2D Tetris 구현한 것이 있고 로직이나 전반적인 게임 자체에는 문제가 없었습니다.  
그러나 폰트 렌더링 부분에서 버그가 있는데, 과거에는 SDL2 확장 라이브러리인 SDL2_TTF를 빌드 후 사용할 수 없었습니다.  
이때, 이 문서를 작성하기 하루 전에 SDL2 확장 라이브러리들을 빌드하는 데 성공하여 SDL2 확장 라이브러리만을 이용해서 다시 만들어보기로 결정했습니다.

<br><br>



## 프로젝트 구현 내용은?

프로젝트 구현에는 다음이 필요합니다.

- 게임 윈도우 창 관리 기능 
- 게임 로그 기능
- 게임 타이머 기능
- 각종 기하 도형, 텍스처, 텍스트 그리기 기능
- 텍스처 로딩
- 트루 타입 폰트 로딩
- Callback 기반의 입력 처리 기능

<br><br>


## 프로젝트 진행 기간은?

프로젝트 진행 기간은 `2022.06.04 ~ 2022.06.11` 약 일주일입니다.  

<br><br>


## 프로젝트 완료 후 다음은?

프로젝트 완료 후 진행할 프로젝트 후보는 다음은 3가지 입니다.  
참고로, 앞으로는 프로젝트 진행 시 반드시 한번에 하나의 프로젝트만 진행하기로 했습니다.  

- `SoftwareRenderer2.0` 프로젝트 
    - `Tetris1.0` 의 구조를 기반으로 `SoftwareRenderer1.0` 의 구조를 개선하고 3D 기능을 추가하는 프로젝트입니다.
- `Tetris2.0` 프로젝트
    - `Tetris1.0` 에서 2D 관련 기능을 3D 기능으로 수정하는 프로젝트입니다.
- 다른 게임 프로젝트
    - 다른 게임 주제를 선정해서 구현하는 프로젝트입니다.

무엇을 할 지는 다음 주에 정하도록 하겠습니다.
