## 조엘 테스트 : 더 나은 코드를 위한 12단계

| 항목 | Y/N |
|:---|:---:|
| 1. 소스코드 관리 시스템을 사용하고 있습니까? | Y/N |
| 2. 한방에 빌드를 만들어낼 수 있습니까? | Y/N |
| 3. 일일 빌드를 사용하고 있습니까? | Y/N  |
| 4. 버그 추적 시스템을 운영하고 있습니까? | Y/N  |
