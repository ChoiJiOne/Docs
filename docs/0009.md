## mkdocs 문서화를 위한 준비 작업

다음 두 가지 설치 필요
```
// 다음은 필수
$ pip install mkdocs

// 다음 아래의 두 가지 중 선택
$ pip install mkdocs-rtd-dropdown
$ pip install mkdocs-material
```


## 문서화 세팅

`mkdocs.yml` 의 내부에 다음 추가
```
theme:
  name: 'material' or `rtd-dropdown`
```
