## 작업을 새로 시작하면 처음보다 훨씬 더 나은 성과를 거두리라는 믿음은 절대로 근거가 없습니다.

작업을 새로 시작하면 처음보다 훨씬 더 나은 성과를 거두리라는 믿음은 절대로 근거가 없습니다. 무엇보다도 버전 1.0에서 함께 일했던 동일한 팀이 존재하지 않기 때문에 실제로는 '더 나은 경험'을 확보할 수 없습니다. 예전 실수 대부분을 되풀이하며 원래 버전에는 없었던 새로운 문제를 불러일으킬 것입니다.
