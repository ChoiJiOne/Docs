## git 커밋 메시지 에디터 변경하기
- git 커밋 메시지 에디터를 변경하기 위해서는 터미널에 다음과 같이 입력한다.

```
$ git config --global core.editor "editor name"
```

- 만약, vim 에디터를 적용하고 싶다면, 다음과 같이 입력한다.

```
$ git config --global core.editor "vim"
```
