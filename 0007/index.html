    32bit와 64bit 레지스터 차이 확인하기

    • 작성자 : 최지원
    • 작성일 : 2022.06.27

    상황

    C언어에서 가장 기본이 되는 "Hello World"를 출력하는 코드를 디스어셈블하여 레지스터를 확인하던 중 32bit와 64bit의 레지스터 상태의 차이를 알고 싶은 상황이다.


    비교를 위한 샘플 코드

    비교를 위한 샘플 코드는 다음과 같다. 

    #include <stdio.h>

int main(void)
{
    printf("Hello World");
    return 0;
}



    32bit 상태의 레지스터 확인

    브레이크 포인트를 걸고 레지스터를 확인할 때, 다음을 확인할 수 있다.
    EAX = 016CA0E0
    EBX = 010C7000
    ECX = 00000001
    EDX = 016CF760
    ESI = 00521023
    EDI = 00521023
    EIP = 00521840
    ESP = 012FFDCC
    EBP = 012FFDE8
    EFL = 00000206



    64bit 상태의 레지스터 확인

    브레이크 포인트를 걸고 레지스터를 확인할 때, 다음을 확인할 수 있다.
    RAX = 0000000000000001
    RBX = 0000000000000000
    RCX = 0000000000000001
    RDX = 00000240D3E2A560
    RSI = 0000000000000000
    RDI = 0000000000000000
    R8  = 00000240D3E30010
    R9  = 000000BF325AF768
    R10 = 0000000000000012
    R11 = 000000BF325AF810
    R12 = 0000000000000000
    R13 = 0000000000000000
    R14 = 0000000000000000
    R15 = 0000000000000000
    RIP = 00007FF740781850
    RSP = 000000BF325AF868
    RBP = 0000000000000000
    EFL = 00000200
