기능 명세 Functional Specification

기능 명세 Functional Specification 는 완전히 사용자 관점에서 제품이 어떻게 동작할 지를 기술합니다.
어떻게 구현했는지는 신경도 쓰지 않습니다.
기능에 대해 이야기하고, 화면, 메뉴, 대화 상자 같은 사용자 인터페이스 부품을 명세합니다.
